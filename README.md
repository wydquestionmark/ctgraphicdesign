# Graphic Design at OWTS static site

## How to use

1. Upload these files to a GitHub repository.
2. Put student images inside the `student-work` folder.
3. Open `script.js` and replace:
   - `YOUR-GITHUB-USERNAME`
   - `YOUR-REPO-NAME`
4. Turn on GitHub Pages in the repository settings.

The gallery uses GitHub's public Contents API to read the `student-work` folder and build a clickable image grid automatically.

Supported image types: png, jpg, jpeg, gif, webp, svg, avif.

## Important

The repository must be public for the browser to read the folder without authentication.
